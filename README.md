# Guessing Game - Java GUI Application

A Java-based GUI application built with Swing.

## How to Get Started

1. Clone the Repository
Don't just download the ZIP! Use Git so we can stay synced. Open your terminal or IDE and run:
`git clone https://github.com/abelinthegit800/GuessingGame.git`

2. Assets & Icons
All images, icons, and audio files are located in the `src/main/resources` folder.

The app uses relative paths, so they should load automatically once the project is opened correctly.


## 💻 Running the App

For VS Code Users

1. **Extensions:** Make sure you have the **"Extension Pack for Java"** installed.
2. **Open:** Go to `File > Open Folder` and select the `GuessingGame` folder.
3. **Run:** Open `MainFrame.java` (or your main class) and click the **Run** button.

### For NetBeans Users
1. **Open:** Go to `File > Open Project`.
2. **Import:** Select the project folder. If you see a Maven or Gradle icon, NetBeans will handle the libraries for you.
3. **Run:** Right-click the project and select **Run** (or press `F6`).

---

## 🛠 Project Structure
* `src/main/java`: Contains all the logic and GUI code (Panels, Frames, etc.).
* `src/main/resources`: Contains the icons and game assets.
